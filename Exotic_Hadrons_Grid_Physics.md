# Grid Physics: Exotic Hadron Compilation Protocol

## Abstract
The Standard Model describes exotic hadrons (tetraquarks, pentaquarks) through complex quantum chromodynamics (QCD) empirical potentials, often treating them as mesonic molecules. **Grid Physics (Simureality)** completely discards this continuous approach. We postulate that quarks are not fundamental particles moving through empty space, but rather **Routing Exceptions (topological defects)** within a discrete 3D Face-Centered Cubic (FCC) lattice. 

Unlike stable crystalline matter that minimizes system debt ($\Sigma K \to \min$), exotic hadrons are "anti-optimal" configurations. They represent violent buffer overflows, demanding significant system resources to isolate conflicting heavy data nodes and prevent topological collapse. This document codifies the strict, deterministic mathematical rules the Matrix (the computational substrate) uses to calculate the final mass of these exotic states.

The total mass of any exotic hadron is strictly calculated as the sum of its "bare" information load and the topological routing lag (hardware overhead). No empirical QCD parameters are required.

### 1.1 The Law of Squares (Data Payload)

**$M_q = m_e \cdot N^2$**

When heavy defects (like $c$-quarks) are physically close on the FCC lattice, they cause extreme topological friction. The Dispatcher injects an empty 3D structural void to isolate them.
* **Cost of 1 Void:** $\mathbf{939.57 \text{ MeV}}$ *(System standard equivalent to the rest mass of an empty nucleon).*

### 1.3 The Vacuum Impedance ($Z_0$)

* **Cost of $Z_0$:** $\mathbf{376.73 \text{ MeV}}$

## 2. The Three Strict Rules of Compilation

When a collision on a particle accelerator clusters heavy defects, the Universe's Dispatcher applies one of three deterministic routing protocols to establish a stable mass. 

### Rule 1: Emergency Isolation (Class A - Single Heavy Conflict)
**Condition:** A single heavy topological defect ($c$) attempts to merge with a closed-shell defect ($s$). Direct contact will cause a kernel panic (divergent density).
**Action:** The Matrix inserts **ONE Isolation Void** between the defects.
* **Formula:** $Mass = \Sigma Payload + 1 \times Void$
* **Experimental Validation ($D_{s0}^*(2317)$):**
    * Composition: $c, \bar{s}$
    * Payload: $1277.50 + 100.16 = 1377.66$ MeV
    * Grid Calculation: $1377.66 + 939.57 = \mathbf{2317.23 \text{ MeV}}$
    * *CERN Measurement:* $2317.8$ MeV ($\Delta = 0.6$ MeV)

### Rule 2: Resonant Bridge (Class B1 - Symmetric Heavy Conflicts)
**Condition:** Two heavy defects (e.g., $c, \bar{c}$) are present, but the surrounding light quarks (e.g., $u, \bar{u}$) form a perfect **symmetric topology** (isosinglet).
**Action:** The symmetry allows the system to save resources. It injects **ONE Isolation Void** and props open a conductive **Vacuum Impedance link ($Z_0$)** across it to bleed the potential.
* **Formula:** $Mass = \Sigma Payload + 1 \times Void + 1 \times Z_0$
* **Experimental Validation ($X(3872)$):**
    * Composition: $c, \bar{c}, u, \bar{u}$
    * Payload: $2555.0 + 4.08 = 2559.08$ MeV
    * Grid Calculation: $2559.08 + 939.57 + 376.73 = \mathbf{3875.38 \text{ MeV}}$
    * *CERN Measurement:* $3871.69$ MeV ($\Delta = 3.6$ MeV)

### Rule 3: Full Rupture (Class B2 - Asymmetric Heavy Conflicts / Pentaquarks)
**Condition:** Two heavy defects are present, but the light quark topology is **asymmetric** (e.g., charged $d\bar{u}$ structures, or an odd number of nodes like in a 5-quark setup). A stable $Z_0$ bridge cannot be established.
**Action:** The Dispatcher performs a complete topological rupture, dividing the cluster by allocating **TWO separate Isolation Voids**.
* **Formula:** $Mass = \Sigma Payload + 2 \times Voids$
* **Experimental Validation (Pentaquark $P_c(4440)$):**
    * Composition: $c, \bar{c}, u, u, d$
    * Payload: $2555.0 + 4.08 + 4.60 = 2563.68$ MeV
    * Grid Calculation: $2563.68 + (2 \times 939.57) = \mathbf{4442.82 \text{ MeV}}$
    * *CERN Measurement:* $4440.3$ MeV ($\Delta = 2.5$ MeV)
* **Experimental Validation ($Z_c(4430)^-$):**
    * Composition: $c, \bar{c}, d, \bar{u}$
    * Payload: $2555.0 + 6.64 = 2561.64$ MeV
    * Grid Calculation: $2561.64 + (2 \times 939.57) = \mathbf{4440.78 \text{ MeV}}$
    * *CERN Measurement:* $4478.0$ MeV inside a broad $181$ MeV peak (Belle 2007 initially reported $4443$ MeV).
